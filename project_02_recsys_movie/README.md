# MovieLens 20M Hybrid RecSys

Данный  проект посвящен построение рекомендательной системе на основе разложения SVD

### Почему это важно?
Проект решает ключевые проблемы современных RecSys:

1. Холодный старт: Использование контентных фичей (например TMDB API) позволяет рекомендовать новые фильмы, у которых еще нет большого числа оценок.
2. Разреженность данных: Комбинация коллаборативных методов с внешними признаками (гибридный подход) повышает точность рекомендаций по сравнению с чистой матричной факторизацией.
3. Включение временных признаков позволяет учесть контекст просмотра.


### Информации о проекте:
#### Ключевой стек и методы:

 1. Python,  Pandas, API, SVD, PCA
 2. EDA и Feature Engineering: визуализации, PCA для снижения размерности 1128 Genome Tags (создание эмбеддингов контента), обогащение данными через TMDB API (бюджет, сборы, популярность), генерация временных фичей.

4. Валидация: Time-based split, метрика ранжирования Precision@10.

### Результат:
Была обучена рекомендательная система фильмов с результатом Precision@10: 0.5438, то есть это значит, что в среднем 5-6 из 10 рекомендованных фильмов реально совпадают с тем, что пользователь оценил в тесте.
